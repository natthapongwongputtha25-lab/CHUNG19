เชื่อมต่อ Google Apps Script

💡 ข้อมูลสำคัญเกี่ยวกับระบบ Apps Script:

เมื่อนำไฟล์ HTML นี้ไปสร้างเว็บแอปใน Google Apps Script หน้าเว็บบอร์ดเทอร์มินัลนี้จะเชื่อมต่อข้อมูลจากตาราง Google Sheet ของคุณได้ทันทีผ่าน API ของ Google Apps Script โดยตรง

ใส่ลิงก์ Web App URL ที่ได้จากขั้นตอนการติดตั้ง Apps Script ด้านขวา

คลังสินค้ากำลังเชื่อมข้อมูลด้วย LOCAL STORAGE

📋 วิธีสร้าง Web App และเอาไปแปะใน Google Apps Script

  1. เปิด Google Sheet ของคุณ
  2. ไปที่ Extensions (ส่วนขยาย) > Apps Script
  3. สร้างไฟล์ใหม่ชื่อ Index.html แล้วคัดลอกโค้ดของไฟล์นี้ทั้งหมดไปวาง
  4. ที่ไฟล์ Code.gs วางสคริปต์นี้เพื่อเปิดแสดงหน้าเว็บ:
function doGet(e) {
  // ระบบป้องกันข้อผิดพลาดกรณีตัวอักษรพิมพ์เล็ก-ใหญ่ไม่ตรงกัน (Index.html vs index.html)
